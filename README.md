# Moving Average Filter for Sensor Data

This project implements a simple moving average (SMA) filter in Python. 

### Specifications:
- **Sampling Rate:** 50 Hz
- **Window Size:** 100 ms (Calculated as 5 samples)
- **Language:** Python 3.9.13

### How to Run:
Just run the script:
`python moving_average.py`
